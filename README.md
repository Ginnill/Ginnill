# Hi 👋, I'm Gustavo Fortunato

## Senior Software Engineer | Generative AI, Payments & Cloud

I'm a Senior Software Engineer with experience building and evolving high-availability applications, payment systems, financial platforms, educational products, and AI-powered solutions.

I started my career working closely with Full Stack development and, over time, began taking on more complex challenges involving software architecture, integrations, cloud infrastructure, databases, performance, scalability, and systems that cannot simply stop working.

Today, I work across Back-End, Front-End, architecture, data, and product, using technology as a tool to solve real business problems.

My work includes:

- Context and conversation history management
- AI-powered data automation
- Integration between LLMs, APIs, databases, and business rules

- Role-Based Access Control
- Reliability, scalability, observability, and automated testing
